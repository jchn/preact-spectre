# Avatar

import Avatar from './Avatar'
import { Playground, PropsTable } from 'docz'
import faker from 'faker'

## Properties

<PropsTable of={Avatar} />

### Default

<Playground>
  <Avatar src={faker.internet.avatar()} alt="avatar" initial="AB" />
</Playground>

### Initial

If a `src` prop is omitted or while the image is loading you can show initials using the `initial` prop:

<Playground>
  <Avatar initial="AB" size="lg" />
</Playground>

### Size

Use the size prop the get different sizes, size should be one of either: `xs`, `sm`, `lg` or `xl`

<Playground>
  <Avatar src={faker.internet.avatar()} alt="avatar" initial="AB" size="xs" />
  <Avatar src={faker.internet.avatar()} alt="avatar" initial="AB" size="sm" />
  <Avatar src={faker.internet.avatar()} alt="avatar" initial="AB" />
  <Avatar src={faker.internet.avatar()} alt="avatar" initial="AB" size="lg" />
  <Avatar src={faker.internet.avatar()} alt="avatar" initial="AB" size="xl" />
</Playground>

### Presence

A presence indicator can be added using the `presence` prop which should be one of: `online`, `busy` or `away`

<Playground>
  <Avatar
    src={faker.internet.avatar()}
    alt="avatar"
    initial="AB"
    size="lg"
    presence="online"
  />
  <Avatar alt="avatar" initial="AB" size="lg" presence="busy" />
  <Avatar
    src={faker.internet.avatar()}
    alt="avatar"
    initial="AB"
    size="lg"
    presence="away"
  />
</Playground>

### Icon

Add an icon by placing a url pointing to an image in the `icon` prop:

<Playground>
  <Avatar
    src={faker.internet.avatar()}
    alt="avatar"
    initial="AB"
    size="lg"
    icon={faker.internet.avatar()}
  />
</Playground>
