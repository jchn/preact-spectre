# Grid

import Grid from './'
import times from './times'
import { Playground, PropsTable } from 'docz'

## Properties

<PropsTable of={Grid} />

### Default

<Playground>
  <Grid>
    <Grid.Columns>
      {times(i => {
        const n = Math.ceil(i / 2) + 1
        const col = i % 2 === 0 ? n : 12 - n + 1
        return (
          <Grid.Col col={col} key={i}>
            <div class="p-2 mb-2 bg-primary">{col}</div>
          </Grid.Col>
        )
      }, 23)}
    </Grid.Columns>
  </Grid>
</Playground>

### Gapless

Remove the gutters by adding the `gapless` prop to `<Grid.Columns />`:

<Playground>
    <Grid>
      <Grid.Columns gapless>
        {times(i => {
          const n = Math.ceil(i / 2) + 1
          const col = i % 2 === 0 ? n : 12 - n + 1

          return (
            <Grid.Col col={col} key={i}>
              <div class="p-2 mb-2 bg-primary">{col}</div>
            </Grid.Col>
          )
        }, 23)}
      </Grid.Columns>
    </Grid>

</Playground>

### Responsive

Make the columns responsive by passing an array to the `col` prop of `<Grid.Col />`:

<Playground>
    <Grid>
      <Grid.Columns>
        {times(i => {
          const n = Math.ceil(i / 2) + 1
          const cols = i % 2 === 0 ? [12, 6, n] : [12, 6, 12 - n + 1]

          return (
            <Grid.Col col={cols} key={i}>
              <div class="p-2 mb-2 bg-primary">{cols.join(',')}</div>
            </Grid.Col>
          )
        }, 23)}
      </Grid.Columns>
    </Grid>

</Playground>
