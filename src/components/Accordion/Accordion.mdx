### Default

The default Accordion component:

### Arrow

Add the `arrow` prop to place arrows in front of the `Accordion.Item`s:

<Playground>
  <Accordion arrow uid="accordion-2">
    <Accordion.Item title="Elements">Element 1</Accordion.Item>
    <Accordion.Item title="Layouts">Layout 1</Accordion.Item>
    <Accordion.Item title="Components">Component 1</Accordion.Item>
  </Accordion>
</Playground>

### Exclusive

Add the `exclusive` prop to have only a single item open at a time

<Playground>
  <Accordion exclusive arrow uid="accordion-3">
    <Accordion.Item title="Elements">Element 1</Accordion.Item>
    <Accordion.Item title="Layouts">Layout 1</Accordion.Item>
    <Accordion.Item title="Components">Component 1</Accordion.Item>
  </Accordion>
</Playground>
