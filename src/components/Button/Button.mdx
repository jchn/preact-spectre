# Button

import Button from './Button'
import { Playground, PropsTable } from 'docz'

### Import

```
import { Button } from '@jchn/preact-spectre
```

## Properties

<PropsTable of={Button} />

### Default

The default button:

<Playground>
  <Button>Default Button</Button>
</Playground>

### Kind

This can be one of `primary`, `success` or `error` and will update the color of the Button:

<Playground>
  <Button kind="primary">Primary Button</Button>
  <Button kind="success">Success Button</Button>
  <Button kind="error">Error Button</Button>
</Playground>

### Size

This can be one of `sm` or `lg`, the default is medium:

<Playground>
  <Button size="sm">Sm Button</Button>
  <Button>Default Button</Button>
  <Button size="lg">Lg Button</Button>
</Playground>

### Loading

Add the `loading` prop for the loading state:

<Playground>
  <Button loading>Loading Button</Button>
</Playground>

### Link

Add the `link` prop to make the Button look like a link:

<Playground>
  <Button link>Link Button</Button>
</Playground>

### Clear

Add the `clear` prop for close/clear actions:

<Playground>
  <Button clear />
</Playground>

### Active

Add the `active` prop to let the Button present itself in the active state:

<Playground>
  <Button active>Active Button</Button>
</Playground>

### Block

Add the `block` prop to let the Button take up the entire width of its container:

<Playground>
  <Button block>Block Button</Button>
</Playground>

### Shape

Use the `shape` prop to set the shape of the Button, this can be either `circle` or `square`:

<Playground>
  <Button shape="circle" intent="primary" />
  <Button shape="square" intent="primary" />
</Playground>
